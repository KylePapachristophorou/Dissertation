Image Implementation:

Image Implementation Continued:
You can make the image a link when clicked by embedding it in a keyboard_arrow_lefta keyboard_arrow_right tag and giving it a href="location" attribute.
keyboard_arrow_left a id="logo-container" href="index.html" class="brand-logo"keyboard_arrow_right keyboard_arrow_left img src="./css/images/index/Logo.svg" alt="Alternate Text" width="350"keyboard_arrow_right keyboard_arrow_left/akeyboard_arrow_right
