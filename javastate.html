(Quick Fact: Javascript is Case Sensitive so be careful!)
Statements use javascript to declare values, an example of this would be to write var A = 5;
Mutliple statements can be chained together if a semi colon is used like this var A = 5; var B = 10;
Statements use opperators and can complete calculations such as var A + var B = var C in this case C would equal 15.
