Data Types

  • arrow_drop_down
    Data Types Overview
    The main 5 data types used in javascript are Var, Int, Double, Boolean and String.
    Int stands for interger, Double includes decimal places, String can be remembered as a string of characters.
    Var is different however, this stands for variable. In most cases Var will acomplish the desired task, however knowing the other data types remains important. Boolean is unlike the other data types and can be thought of as true or false.
  • arrow_drop_down
    Number Data Types
    Whole Numbers
    Intergers are declared with the "Int" data type, these will only be full numbers and do not include decimal places.
    Numbers With Remainders
    Doubles are declared the same unlike intergers that are shortened to int, this data type is used for numbers with remainders such as "4.5" or "15.4".
  • arrow_drop_down
    Text Data Types
    Text Data Types Text data can easily be stored with either a String or a Var data type, the String can be easily remembered as a string of characters which includes numbers.
    Boolean is easily remembers as true or false, this is mostly used in for example ( A > B) as A is either greater than B in which case it is true or B is greater and it is false.
