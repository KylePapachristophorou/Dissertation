Development Tools

  • arrow_drop_down
    Web Browser Tools
    Google Chrome is arguably the best browser to use for web development, with many features and backwards compatibility it is one of the most widely used browsers avaliable. Chrome includes the developer tools on F12 which enables multiple device testing and the ability to click an element on screen to closer inspect it such as how it is being styled.

    One feature often overlooked is the ability to use this on other websites to learn about new and interesting techniques, this can help improve web development skills quickly when used along side templates and video tutorials.
  • arrow_drop_down
    Editor Options
    Many code editors include features to improve the design experience, some include spelling checks while others such as Brackets include the ability to search for a phrase and replace it in all files in a selected folder!
    This is incredibly useful when updating a sites navigation and quick fixing a error made across multiple pages.
